# 缝纫原理
- 缝纫的核心是**用线将两块（或多块）布料永久连接**，本质是通过特定的**线迹结构**，让针线在布料间形成牢固的锁合。
- [缝纫机的天才设计](https://www.bilibili.com/video/BV1Fg4y1r7wk/?spm_id_from=333.337.search-card.all.click&vd_source=10d88f7c37a300eb9dc9542bd5580133)
## 手缝原理
- 手缝的核心动作只有 3 步，本质是 “让线在布料两侧交替穿梭，形成可受力的线结 / 线迹”：
	1. **穿针引线**：把线穿过针眼，线尾打一个止线结（防止线从布料中滑脱）；
	2. **针线穿梭**：将针尖从布料正面扎入、反面穿出（或反之），拉出线段；再调整针尖位置，从反面扎回 正面，重复这个 **穿入 - 穿出** 的动作，让线在布料两面形成连续的线段；
	3. **形成固定**：通过不同的**穿梭规律**（比如**平针**、**回针**、**锁边针**），让线段之间形成相互拉扯的力。

- 缝纫机原理是围绕 “如何高效模仿并优化手缝动作” 设计的，核心依赖 **机针 + 梭芯（或旋梭）+ 送布牙 + 压脚** 四个部件的协同工作，最终形成牢固的 **锁式线迹**（最常用、最基础的线迹，占日常缝纫的 90% 以上）。
1. **核心前提：双线系统**
	- **面线**：从缝纫机顶部的 **线轴** 引出，穿过 **穿线孔**、**夹线器**（控制线的松紧），最终穿入 **机针** 的针眼；
	- **底线**：绕在 **梭芯**（一个小圆柱状线轴）上，放在缝纫机的 **梭壳**（或旋梭）里，隐藏在缝纫机工作台下方（或机头内部）。
	- *两根线的作用类似 “拉链的两个齿”：面线从布料正面往下穿，底线从布料反面往上勾，两者在布料中间 “交织锁合”，形成的线迹既牢固又不易脱线（手缝单根线若断裂，整段线迹会散，而双线锁合只要两根线不同时断，线迹就不会散）。*
2. **关键步骤：4 个部件协同，完成一次 “锁式线迹”**
	- 缝纫机的工作是循环往复的，每一个循环对应 一个完整的锁式线迹，具体分 5 步：

| 步骤             | 部件动作                                                                                             | 线的状态（核心）                                           |
| -------------- | ------------------------------------------------------------------------------------------------ | -------------------------------------------------- |
| 1 机针向下穿刺       | 电机带动机针（上下运动），针尖带着 “面线” 从布料**正面扎入**，穿过布料到达**反面**                                                  | 面线随针穿过布料，在反面留下一个 “线环”（机针上有个 “线槽”，线会顺着线槽形成环）        |
| 2 梭芯 / 旋梭勾住面线环 | 机针到达最低点时，隐藏在下方的 “梭芯 / 旋梭”（旋转运动）刚好转到机针旁，用 “梭尖” 勾住机针留下的 “面线环”                                      | 底线在梭芯上，旋梭勾住面线环后，会带着面线环 “绕底线转一圈”（类似用绳子绕另一根绳子打个结）    |
| 3 机针向上回升       | 机针开始向上运动，从布料反面抽回正面；同时，旋梭继续旋转，把 “绕完底线的面线环” 逐渐收紧                                                   | 面线环被拉到 “布料中间层”（不是正面也不是反面），与底线初步交织                  |
| 4 送布牙带动布料移动    | 当机针抽回正面、线环开始收紧时，缝纫机工作台下方的 “送布牙”（像小牙齿一样的部件）向上顶起布料，同时向前推动一小段距离（距离由 “针距旋钮” 控制，比如 1cm 推 3 针就是针距 3mm） | 布料移动后，机针下次穿刺的位置会错开，保证线迹是 “连续的直线”（而不是在同一个点反复穿刺）     |
| 5 夹线器收紧线迹      | 机针回到最高点时，缝纫机顶部的 “夹线器”（类似两个小夹子）会适度夹紧面线，把 “布料中间的线环” 彻底收紧，形成一个 “平整、不松动” 的锁式线迹                       | 至此，一个完整的锁式线迹完成；紧接着，机针再次向下，重复步骤 1-5，形成连续的线迹，把布料连接起来 |
3. **其他线迹的原理（本质是 “基础动作的变形”）**
	- 除了最常用的 **锁式线迹**（用于缝合布料、拼接），缝纫机还能做 **包边线迹**（之字形迹）、 **钉扣迹** 等，原理其实是在锁式线迹的基础上，增加了 **机针左右摆动** 或 **送布牙前后左右移动** 的动作：
		- **包边迹**（用于锁边、加固布边防毛边）：、
			- 机针上下运动时，会同时左右摆动（摆动幅度可调），送布牙正常向前送布，最终面线和底线交织成之字形，能把布料边缘包起来。
		- **钉扣迹**：
			- 送布牙不送布（布料不动），机针在同一个小范围内左右 / 前后摆动，反复穿刺布料和纽扣的扣眼，用线把纽扣固定在布料上。
## 总结
1. **本质是 “线的交织”**：无论是手缝还是机缝，最终都是靠 “线在布料间的规律交织” 形成固定。
2. **双线锁合是核心**：缝纫机的 “面线 + 底线” 锁合结构，是它比手缝更牢固、高效的关键，也是所有基础缝纫的核心逻辑。
3. **部件协同是关键**：机针（带面线上下）、梭芯（带底线勾线）、送布牙（带布料移动）、压脚（压稳布料不打滑）四个部件的同步动作，才实现了 “连续、均匀、牢固” 的缝纫。

# 家用机 & 工业机
- [家用缝纫机和工业平车的区别](https://www.bilibili.com/video/BV1AT4y1T7A5/?spm_id_from=333.337.search-card.all.click&vd_source=10d88f7c37a300eb9dc9542bd5580133)
- [【21天从零开始学做衣】Day5新手如何选缝纫机](https://www.bilibili.com/video/BV1R84y167Ut/?spm_id_from=333.1387.homepage.video_card.click&vd_source=10d88f7c37a300eb9dc9542bd5580133)

| 类别         | 家用机                                                                         | 工业机                                                                                                              |
| ---------- | --------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| **设计定位**   | 满足家庭日常轻度缝补需求                                                                | **高速度**、**高稳定性**、**功能专一性强**                                                                                      |
| **功能侧重**   | 功能多样：基础缝、锁边、钉扣、绣花、曲折缝等（“一机多用”）                                              | 功能单一：多为 “单功能机型”（如专门缝直线的平缝机、专门锁边的包缝机、专门缝皮革的高头机），专注提升单一工序效率。                                                       |
| **体积与重量**  | **小巧轻便**，机身多为塑料 / 轻量化金属，重量 **5-15kg**，可放桌面                                  | 体积大、重量沉，配有专用工作台（桌）                                                                                               |
| **缝纫速度**   | 低速为主，最高速度 800-1500 针 / 分钟，可手动调节快慢，效率较低                                      | 高速运行，最高速度 2000-5000 针 / 分钟（部分机型可达 8000 针），速度多为固定档位。效率高，如后续有接单等商业化发展需求，尽头就是工业机                                    |
| **电机功率**   | 小功率电机（30-100W），适合薄 / 中厚布料（棉、麻、薄牛仔）                                          | 大功率电机（150-500W），可轻松应对厚料（多层帆布、皮革、厚牛仔）                                                                             |
| **线迹系统**   | 现在的家用机线迹种类丰富多样，调节简单，贵多不贵精                                                   | 按需求匹配线迹：**平缝机（锁式）**、包缝机（包边锁式）、绷缝机（覆盖线迹）等，线迹张力调节更精细。<br>比如平缝机只能车直线，但大部分衣物、包等日常缝纫都是车直线                             |
| **耐用性与维护** | 使用寿命 5-10 年（日常家用频率），维护简单（清理线头、换梭芯）。<br>但因为功能较多，内部零件标准化通用化程度相对低，出了问题大概率得返厂维修 | 24 小时连续工作设计，使用寿命 10-20 年，需定期专业维护（换机油、调旋梭、磨针板），<br>发展成熟，内部零件通用化程度高，机器牌子不同但零件都通用，相对好维护修理                           |
| **入手方式**   | 在淘宝、京东且售后优秀的店铺购买全新大牌机。<br>不建议收闲鱼个人买家的机器                                     | 1 可在淘宝购买全新，有售后<br>2 由于工业机较为稳定，不易坏，可在淘宝购买较新的、调试好的二手机，找有售后的店铺                                                      |
| **出手方式**   | 挂闲鱼上相对好卖                                                                    | 体积太大，一般本地自提、折价回收                                                                                                 |
| **价格参考**   | **SINGER 6805C（胜家）** ：1900                                                  | **Brother 7250（兄弟）** ：4700<br>**Brother 7300（兄弟）** ：5700<br>**JUKI 9000C（重机）** ：6700<br>淘宝调试好有售后的二手价格在 2000~3500 |
1. 建议先试试**玩具机**，了解并掌握基础缝纫流程 → 再尝试**家用机** → 如后续有更专业的需求，可选择**工业机**。 
	- *假如热情减退家用机相对好出手*
	- *可能由于功能较多，内部构造复杂，体积又较小，家用机的故障率挺高，较好的家用机价格和工业机差不多甚至更贵（某些方面有点类似笔记本电脑和台式电脑？）*
	- *由于上述兄弟、重机的工业机型号都为进口，较少参加优惠，全新可以讲讲价，或者入手二手*
2. 也可先试试**玩具机**，了解并掌握基础缝纫流程 → 直接一步到位选择适配的**工业机**。
- **看到一则评论：** **工业平车** + **家用缝纫机**（来实现锁眼、人之线迹等基础线迹功能）＋ **四线锁边工业机**（这个还没了解，应该是专业性较强的锁边机）
